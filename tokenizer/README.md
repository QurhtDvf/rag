
# minbpeトークナイザーの実装

このリポジトリでは、Byte Pair Encoding (BPE) アルゴリズムに基づいたシンプルなトークナイザー `BPETokenizer` を独自に実装しています。BPEは、自然言語処理（NLP）タスクにおいて広く利用されているサブワードトークナイゼーション手法の一つです。

## BPE (Byte Pair Encoding) とは？

BPEは、データ圧縮の分野で生まれたアルゴリズムで、自然言語処理の文脈では、未知語（Out-Of-Vocabulary, OOV）問題への対処や、語彙サイズを管理するために利用されます。特に、大規模なテキストデータセットから効率的なサブワード単位の語彙を構築するのに適しています。

### BPEの仕組み

1.  **初期語彙の構築**: まず、テキストを構成する最小単位（文字、または本実装のようにバイト）から初期語彙を構築します。本実装では、UTF-8エンコードされたテキストの全バイト（0-255）を初期トークンとしています。
2.  **最も頻繁なペアのマージ**: テキスト内で最も頻繁に出現する隣接するトークンのペアを見つけます。
3.  **新しいトークンの生成**: 見つかったペアを新しい単一のトークンとして語彙に追加し、テキスト内のすべての出現箇所を新しいトークンで置き換えます。
4.  **繰り返し**: 目標とする語彙サイズに達するか、あるいはそれ以上マージ可能なペアがなくなるまで、ステップ2と3を繰り返します。

このプロセスにより、頻繁に出現する単語やサブワードが徐々に結合され、より長い意味のあるトークンが形成されます。例えば、「programmer」という単語は、「p」「r」「o」「g」「r」「a」「m」「m」「e」「r」から始まり、最終的には「program」「mer」といったサブワードに、あるいは「programmer」全体にマージされる可能性があります。

## `BPETokenizer` クラスの機能

本実装の `BPETokenizer` クラスは、以下の主要な機能を提供します。

*   **`__init__(self)`**: トークナイザーの初期化を行います。語彙、マージ規則、およびその逆マッピングを格納する辞書を準備します。
*   **`train(self, text, vocab_size)`**: 与えられたテキストデータ (`text`) を基に、指定された語彙サイズ (`vocab_size`) に達するまでBPEアルゴリズムを実行し、語彙とマージ規則を学習します。
*   **`_get_pair_counts(self, tokens)`**: トークン列内の隣接するペアの出現頻度を計算します。
*   **`_merge_pair(self, tokens, pair, new_token_id)`**: 特定のペアを新しいトークンIDで置き換えた新しいトークン列を返します。
*   **`encode(self, text)`**: 訓練済みの語彙とマージ規則を使用して、入力テキストをトークンIDのリストに変換します。訓練時に学習されたマージ規則を繰り返し適用することで、テキストをサブワードに分割します。
*   **`decode(self, tokens)`**: トークンIDのリストを元のテキスト文字列にデコードします。各トークンIDに対応するバイト列を結合し、それをUTF-8でデコードして文字列に戻します。
*   **`get_vocab_list(self)`**: 現在の語彙を、デコード可能な文字列のリストとして返します。

## 日本語マルチバイト文字の処理

本実装の`BPETokenizer`は、テキストをUTF-8バイト列にエンコードしてからBPEアルゴリズムを適用します。これにより、日本語のようなマルチバイト文字（例えば「こ」はUTF-8で3バイト）はバイトのシーケンスとして扱われます。

BPEは、これらのバイトを個別の要素として認識し、最も頻繁に出現するバイトペア（またはバイトとマージされたトークンのペア）をマージして新しいトークンを生成します。このため、頻繁に出現する日本語の文字のシーケンスや単語が、効率的に1つのトークンとして学習されます。

例えば、「こんにちは、世界！」というテキストを訓練した場合、語彙には「こ」、「ん」、「に」、「ち」、「は」といった個別の文字に対応するトークンだけでなく、「こんにちは」、「世界！」、さらにはそれらが連結された「こんにちは、世界！」などの長いフレーズに対応するトークンも生成されることがあります。

## ファイル構成

*   `bpe_tokenizer.py`: `BPETokenizer` クラスの定義が含まれています。
*   `bpe_tokenizer_example.py`: `bpe_tokenizer.py` をインポートし、`BPETokenizer` クラスを利用してテキストの訓練、エンコード、デコードを行うサンプルコードが含まれています。

## 使用方法

1.  `bpe_tokenizer.py` をプロジェクトにコピーします。
2.  以下のように `BPETokenizer` をインポートして使用します。

    ```python
    from bpe_tokenizer import BPETokenizer

    # トークナイザーのインスタンス化
    tokenizer = BPETokenizer()

    # サンプルテキスト
    sample_text = "こんにちは、世界！これはBPEトークナイザーのテストです。"

    # 語彙サイズを設定
    vocab_size = 300

    # トークナイザーを訓練
    tokenizer.train(sample_text, vocab_size)

    # エンコード
    encoded_tokens = tokenizer.encode(sample_text)
    print(f"Encoded Tokens: {encoded_tokens}")

    # デコード
    decoded_text = tokenizer.decode(encoded_tokens)
    print(f"Decoded Text: '{decoded_text}'")

    # 語彙リストの取得
    vocab_list = tokenizer.get_vocab_list()
    print(f"Vocab Size: {len(vocab_list)}")
    ```

`bpe_tokenizer_example.py` ファイルには、この一連の処理をまとめた完全なサンプルコードが記述されています。
